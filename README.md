# Dion Italy E-Commerce Storefront

This repository contains the codebase for [Dion Italy](https://dionitaly.com), an online boutique specializing in premium, handmade Italian shoes. 

**Built and designed by Akram Moni.**

## 🛍️ Product Catalog
The storefront features a curated selection of footwear, including:
- **Dion Italy Loafers:** Classic slip-on designs.
- **Retro Sneakers:** Vintage-inspired athletic footwear.
- **High-Top Sneakers:** Premium leather and custom-style high-tops.
- **Slides & Specialized Footwear:** Comfortable casual options.

## 🚀 Key Features
- **Custom Design & UX:** Visually tailored interface built from the ground up to reflect a premium luxury brand aesthetic.
- **Direct-to-Consumer (DTC) Sales:** Fully integrated shopping cart and secure checkout experience.
- **Dynamic Catalog:** Supports complex product variants, such as color and size options, for collections like the *Arcobaleno TD* and *Vibrazione HTWS*.
- **Responsive UI:** Fully optimized for mobile, tablet, and desktop browsing.

## 🛠️ Technology & Stack
- **Design & Architecture:** Akram Moni
- **Platform:** WordPress / WooCommerce
- **Frontend:** Responsive layout utilizing custom CSS/JS for smooth transitions and rich product presentation.
- **Payments:** Secure integrated gateway for global transactions.

## 📦 Getting Started
To run or contribute to this project:
1. Clone the repository.
2. Ensure you have a local environment set up with PHP and MySQL (compatible with WooCommerce).
3. Import the database schema and upload the theme/plugin assets.
